# LifeTrack – AI-Powered Productivity Assistant

## 🧠 About the Project
LifeTrack is a personal productivity assistant that I built to help manage daily tasks, schedules, and commitments in a smarter way.

Most productivity apps today depend heavily on cloud services, which raises privacy concerns. So the idea behind LifeTrack was simple — **create a system that works completely offline**, keeps user data private, and still feels intelligent.

It uses AI (NLP models) to understand tasks, organize them, and provide useful insights — all running locally on your system.



## ✨ What It Can Do
- Add tasks using text or voice  
- Automatically understand and categorize tasks using NLP  
- Generate a daily schedule based on priorities  
- Show productivity insights through a dashboard  
- Give reminders and track commitments  
- Run fully offline (no internet or API needed)



## ⚙️ Tech Stack
**Backend:** Flask (Python)  
**Frontend:** HTML, CSS, Bootstrap / Tailwind  
**Database:** SQLite (local storage)  

**AI/ML:**
- HuggingFace Transformers  
- spaCy  
- Whisper / Vosk (for voice input)  

**Visualization:**
- Plotly / Matplotlib  



## 🏗️ How It Works (Simple Flow)

User Input (Text/Voice)  
→ AI processes the task  
→ Task gets stored locally  
→ Scheduling engine organizes it  
→ Dashboard shows insights
